Tea Tracker

Introduction

I like tea. Whenever I have a cup, for a while I tracked it (with a fairly decent level of accuracy) using a little HTML5 app I built and wrapped using PhoneGap. This page was originally built to play with AngularJS.

I've shut the project down now, and this site houses the data that I collected between the 1st November 2013 and 13th November 2015. The data is available under a Creative Commons attribution for you to use, should you wish (CC BY 4.0).
